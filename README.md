# Digital Closet App - Project 1

## Overview

Digital Closet App is a frontend web application that allows users to organize their clothing items, browse their personal closet, and generate outfit combinations.

This project was developed as Project 1 of the DecodeLabs Internship Program using HTML, CSS, and JavaScript.

## Features

### User Interface

* Responsive design
* Modern and clean layout
* Navigation between application pages

### Closet Management

* Add clothing items
* Upload clothing images from the user's device
* View clothing items in a closet grid
* Delete clothing items

### Outfit Generator

* Generate random outfits from available clothing items
* Display selected top, bottom, shoes, and accessories

### Custom Outfit Builder

* Select clothing items manually
* Create personalized outfit combinations

### AI Try-On Page (UI Prototype)

* Upload user photo
* Select clothing items from closet
* Placeholder interface for future AI outfit visualization

### Additional Features

* Category filtering
* Search functionality
* Local browser storage persistence
* Mobile-friendly design
